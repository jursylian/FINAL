# Ручное тестирование

Навигация: [README.md](README.md) → [02-phases.md](02-phases.md)

## Auth
- Регистрация с корректными данными → 201 + token.
- Регистрация с дублирующим email/username → 409.
- Логин с неверным паролем → 401.
- `GET /auth/me` без токена → 401.

## Профиль
- Просмотр профиля `GET /users/:id` → без `password`.
- Редактирование профиля владельцем → 200.
- Редактирование чужого профиля → 403.
- Загрузка аватара через FormData → 200 и обновление `avatar`.

## Посты
- Создание поста без `image` → 400.
- Создание поста с изображением → 201.
- Лента возвращает пагинацию.
- Удаление поста владельцем → 204.

## Лайки и комментарии
- Два лайка подряд → второй снимает лайк.
- Toggle like на комментарий `POST /comments/:id/like` работает.
- Комментарий пустой → 400.
- Комментарии возвращаются постранично.

## Поиск и Explore
- Поиск по `q` → корректные пользователи.
- Explore возвращает случайный порядок.

## Подписки
- Follow/unfollow работает и не дублируется.
- Списки followers/following корректны.

## Уведомления
- Создание уведомления на like/comment/follow/like_comment.
- `GET /notifications/unread-count` возвращает корректный счётчик.
- Пометка прочитанным меняет `read`.
- Удаление уведомления `DELETE /notifications/:id` работает.

## Smoke‑тесты
- Регистрация → логин → просмотр ленты → создание поста → лайк → комментарий.
- Выход/перезагрузка → сохранение токена и повторный вход.
